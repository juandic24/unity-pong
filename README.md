# 🎮 Unity Pong

A simple Pong game built with Unity, personal project to learn and polish Unity fundamentals, still have few bugs and things to improve.

## 🕹️ Play the game
👉 https://juandc25.itch.io/modern-pong

## 🚀 Features
- AI opponent or Multiplayer
- Easy, Medium and Hard difficulties
- Increasing ball speed
- Classic Pong mechanics

## 🎮 Controls
- W / S → Move paddle player 1
- ⬆️/⬇️ → Move paddle player 2 (multiplayer)

## 🛠️ Tech
- Unity
- C#
- Pixel art by my own and my girlfriend with libresprite
- Music made with beepbox
